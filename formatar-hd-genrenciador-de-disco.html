Como Formatar HD com Gerenciamento de Disco?


Passo 1: Clique com o botão do mouse em Este Computador, escolha Gerenciar > Gerenciamento de Disco.


Passo 2: Clique com o botão direito do mouse no disco rígido e selecione a opção Formatar.



Passo 3: No campo "Rótulo do volume", digite um nome descritivo para a unidade.


Passo 4: Use o menu suspenso "Sistema de arquivos" e selecione NTFS (recomendado para o Windows 10).


Passo 5: Use o menu suspenso "Tamanho da unidade de alocação" e selecione a opção Padrão.


Passo 6: Marque a opção Executar uma formatação rápida.



Dica rápida: Ao executar uma formatação rápida, a unidade é limpa mais rapidamente...


Depois de concluir as etapas, o sistema formatará a partição selecionada na unidade e você poderá começar a usá-la para armazenar arquivos.


Como Formatar HD no Windows Explorer?


Passo 1: Vá para "Este Computador" e abra o Windows Explorer, clique com o botão direito do mouse no disco rígido e escolha "Formatar".


Passo 2: Especifique o sistema de arquivos e defina um novo rótulo para o disco rígido. Clique em "Iniciar" e "OK" para confirmar a formatação.
